# 概要
dorisol1019が欲しい機能を詰め込んだフリーの音楽プレーヤーソフトです

# 機能
再生・停止
一時停止・一時停止解除
midiファイルは#CC111ループ対応
リストの順番通りの再生/シャッフル再生

# 動作要件
１：OSがWindows Vista SP2以上であること
２：.Net Framework 4.5がインストールされていること

１を満たしていて動作しない場合、.Net Framework 4.5をインストールしてください

# 注意事項
大きいサイズ(数十MB)のファイルを読み込むと、ソフトが異常停止するおそれがあります
　
本ソフトが起動できなくなりますので
同梱の拡張子が.dllや.configのファイルはアンインストール時以外、絶対に削除しないで下さい

「#CC111が最後の発音のあとにある」midiファイルはループ再生しませんので気をつけて下さい

# 使用方法
１：Playlistフォルダに楽曲を入れます
２：本ソフトを起動します
３：気分でループや音量の設定をします
４：再生ボタンを押して楽曲を楽しみます
５：止めたければ停止ボタンを押します
６：「ファイル(F)」→「フォルダを開く(O)」で楽曲を読み込むフォルダを変更することができます
７：現在開いているフォルダに新たに楽曲を入れた場合、「ファイル(F)」→「再起動」でリストを更新することが出来ます

# 削除
このフォルダごとゴミ箱にぶち込んでください
